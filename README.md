# Repository Didattica
## Build a Large Language Model (From Scratch) - Sebastian Raschka
Questa repository contiene l'implementazione pratica del libro **"Build a Large Language Model (From Scratch)"** di Sebastian Raschka.
<img width="686" height="386" alt="image" src="https://github.com/user-attachments/assets/4859fb98-cb54-4eae-8d78-16df2b3ef379" />

L'intero progetto è strutturato in una serie di **Jupyter Notebook, interamente commentati in italiano**, pensati per un apprendimento attivo. Invece di limitarsi a leggere la teoria, qui puoi eseguire il codice, modificare i parametri e sperimentare direttamente con ogni componente di un LLM.

## Cosa troverai all'interno
* **Apprendimento Interattivo:** Ogni notebook è eseguibile e ti permette di testare i concetti in tempo reale.
* **Contenuti in Italiano:** Tutte le spiegazioni e i commenti al codice sono tradotti per facilitare la comprensione.
* **Percorso Step-by-Step:** I notebook seguono fedelmente la struttura del libro, guidandoti nella costruzione di un LLM completo partendo da zero.
